# Fake News Detection System

Simple AI-powered fake news detector using LLM.

## Quick Start

```bash
streamlit run app_llm_only.py
```

## What It Does

- Enter a news claim
- AI analyzes if it's TRUE or FALSE
- Shows verdict with explanation
- Beautiful cyberpunk UI

## Requirements

```bash
pip install streamlit groq
```

## API Key

Your Groq API key is already configured in `app_llm_only.py`.

## Files

- `app_llm_only.py` - Main Streamlit app (USE THIS)
- `app.py` - Complex version with evidence retrieval (optional)
- `src/` - Source code modules
- `.kiro/specs/` - Project specifications
